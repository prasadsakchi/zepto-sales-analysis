## Business Objective
- Analyze order and product performance  
- Identify demand patterns

## Dataset
- Zepto transactional dataset  
- Includes order, product, quantity, and pricing information  

## Tools Used
- SQL (Data Cleaning & Analysis)

## Analysis Performed
- Order-level analysis to understand demand volume  
- Product-level analysis to identify top-performing items  
- Price and quantity-based analysis  
- Time-based trend analysis (where applicable)  

## Key Insights
- A small number of products contribute a large share of orders  
- Certain price ranges show higher order frequency  
- Order patterns vary across time  

## Key Learnings
- Translating business questions into SQL queries  
- Using aggregations and filters for analysis  
- Answering business problems without dashboards
